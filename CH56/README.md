# Reverse Linked List

## Difficulty: Easy

## Description

Given a vector representing a linked list, reverse the list and return the result.

(In real interviews, this problem uses linked lists. Here we use vectors to simplify the Rust implementation.)

## Examples

**Example 1:**
```
Input: [1, 2, 3, 4, 5]
Output: [5, 4, 3, 2, 1]
```

**Example 2:**
```
Input: [1, 2]
Output: [2, 1]
```

**Example 3:**
```
Input: []
Output: []
```

## Constraints

- `0 <= list.len() <= 5000`
- `-5000 <= list[i] <= 5000`

## Hint

You can solve this iteratively or recursively. Try implementing both approaches.
